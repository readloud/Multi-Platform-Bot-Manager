# Website Traffic Viewer Auto Backlink

1. **Install dependencies:**
```bash
pip install -r requirements.txt
```

2. **Setup database:**
```bash
python setup.py
```

3. **Run server:**
```bash
python main.py
```

4. **Access dashboard:**
```
http://localhost:5000
login: admin / password
```

5. **Add tracker to your website:**
```html
<script src="http://your-server:5000/static/tracker.js"></script>
```
